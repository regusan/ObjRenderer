# Features
* Objファイル読み取り
* PPM画像出力
# Requirement
* make
* g++(C++17)
* Eigen

# Installation
```bash
apt install make
apt install g++
apt install libeigen3-dev

git clone https://github.com/regusan/ObjRenderer.git
```

# Usage
```bash
make release
./renderer_release.out models/room.obj
```
